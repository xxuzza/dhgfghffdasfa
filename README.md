# VPN Telegram Bot для 3X-UI

Telegram бот для автоматической продажи VPN подписок с интеграцией 3X-UI панели.

## Возможности

- ✅ Автоматическое создание VPN конфигураций
- ✅ Управление подписками (срок действия, трафик)
- ✅ Интеграция с 3X-UI API
- ✅ База данных пользователей и подписок
- ✅ Несколько тарифных планов
- ✅ Админ панель

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните:
```bash
cp .env.example .env
```

4. Настройте `.env`:
- `TELEGRAM_BOT_TOKEN` - токен от @BotFather
- `XUI_PANEL_URL` - URL вашей панели 3X-UI
- `XUI_USERNAME` и `XUI_PASSWORD` - данные для входа
- `ADMIN_ID` - ваш Telegram ID

## Настройка 3X-UI

1. Убедитесь что API включен в настройках панели
2. Создайте inbound (VLESS/VMess) для клиентов
3. Запомните ID inbound (по умолчанию 1)

## Запуск

```bash
python bot.py
```

## Настройка тарифов

Отредактируйте `config.py`, секцию `PLANS`:

```python
PLANS = {
    '1_month': {'name': '1 месяц', 'days': 30, 'price': 200, 'traffic_gb': 100},
    # добавьте свои тарифы
}
```

## Интеграция платежей

Для реальных платежей нужно добавить:
- YooMoney (ЮMoney)
- Qiwi
- Криптовалюты
- Или другой провайдер

Замените функцию `process_payment()` в `bot.py` на реальную обработку платежей.

## Команды бота

- `/start` - Главное меню
- `/stats` - Статистика (только для админа)
